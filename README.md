# Dario AI

Dario AI is a German Windows AI assistant.

Dario AI was developed to provide a local personal AI assistant on the user's own computer. The goal is to create an assistant that can speak, listen, execute simple tasks, analyze the computer, and depending on the version, understand what is visible on the screen.

Dario AI is currently designed for German. There is currently no English version of the assistant itself.

## Important Notice

Dario AI is not open-source software.

The source code is private and is not published in this repository. This repository mainly serves as the official information, documentation, and download page for Dario AI.

## Main Features

Depending on the version, Dario AI provides different features.

These may include:

* German AI assistant
* voice control
* voice output
* local AI through Ollama
* hardware diagnosis
* screen recognition
* graphical user interface
* terminal version
* installer
* different editions for different computers
* status displays
* startup and diagnosis checks

## Editions

Dario AI currently has three main editions:

* Dario PRO
* Dario
* Dario Lite

### Dario PRO

Dario PRO is the most advanced edition of Dario AI.

This version is designed for stronger computers and includes the most features. It is intended for users who want to use Dario AI as fully as possible.

### Dario

Dario is the standard version.

This version is designed for normal use and includes the most important Dario AI features without being focused on maximum performance or every advanced function.

### Dario Lite

Dario Lite is the lighter version of Dario AI.

This version is designed for weaker computers and should require less performance. It is useful if the computer does not have enough power for the larger editions.

## UI Version and Terminal Version

Depending on the installation, Dario AI can be used as a UI version or a terminal version.

### UI Version

The UI version includes a graphical user interface. It is designed for users who want to use Dario AI more easily and prefer a modern visual interface.

### Terminal Version

The terminal version runs inside a console window. It is simpler and is mainly useful for diagnosis, testing, or users who prefer a direct technical view.

## Local AI

Dario AI can use local AI models through Ollama.

This allows many AI features to run directly on the user's own computer. This is important for privacy, speed, and independence from external AI websites.

Depending on the selected model and computer performance, response speed may vary.

## Voice Control

Dario AI can react to voice commands.

The user can speak to Dario through a microphone and give commands. Dario can then generate an answer and respond using text-to-speech.

The quality of voice recognition depends on the microphone, background noise, pronunciation, and the speech recognition system used.

## Screen Recognition

Depending on the version, Dario AI can analyze the screen.

This feature can be used to recognize what is visible on the user's screen. It can help Dario understand what the user is looking at when the user asks what is currently visible.

This feature may use screenshots, which are only intended to be used for the active analysis.

## Hardware Diagnosis

Dario AI checks the computer hardware information.

This may include:

* operating system
* CPU
* RAM
* GPU
* microphone
* AI model check

This feature helps determine whether the computer is suitable for Dario AI.

## System Requirements

The exact requirements may vary depending on the selected edition (You can read everything in the installer).

### Minimum Requirements

* Windows 10 22H2 or Windows 11
* 64-bit operating system
* at least 16 GB RAM
* multi-core CPU
* graphics card
* enough free storage space
* microphone or headset for voice features
* Ollama for local AI features

### Recommended Requirements

* Windows 11
* 32 GB RAM
* modern multi-core processor
* dedicated graphics card
* SSD storage
* good microphone or headset
* stable internet connection for setup, downloads, and updates
